# Vértice Imobiliária & Construções

Site institucional desenvolvido para apresentar a Vértice Imobiliária & Construções com uma proposta visual premium, navegação responsiva e foco na comunicação de empreendimentos, serviços e posicionamento de marca.

## Visão geral

O projeto foi construído com React, Vite e TypeScript para oferecer uma base moderna, performática e simples de evoluir ao longo do tempo.

## Tecnologias

- React 19
- Vite 6
- TypeScript
- Tailwind CSS 4
- React Router DOM

## Como rodar localmente

Pré-requisito: Node.js

1. Instale as dependências com `npm install`
2. Inicie o ambiente local com `npm run dev`
3. Acesse `http://localhost:3000`

## Scripts disponíveis

- `npm run dev`: inicia o servidor de desenvolvimento
- `npm run build`: gera a versão de produção
- `npm run preview`: executa a pré-visualização do build
- `npm run lint`: valida o TypeScript
- `npm run clean`: remove a pasta `dist`
